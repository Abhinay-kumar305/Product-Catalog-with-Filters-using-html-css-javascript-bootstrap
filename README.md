# Product Catalog with Filters

## Overview
A responsive Product Catalog UI built with HTML, CSS, Bootstrap, and JavaScript. Features dynamic product filtering, search, price-range selection, hover animations, and an optional wishlist using localStorage.

## Features
- Multi-page layout: Product List, Product Details, About
- Product cards grid with hover effects
- Category and price-range filters (JS)
- Search functionality (JS)
- Wishlist button (localStorage optional)
- Fully responsive design
